# BuyerOrder

1. First extract the folder and place it into the local server.
2. Make a virtual host like "buyer.test" for this folder.
3. Create a database and provide the connection details in "Config/Database.php".
4. Please Execute the dump from the db folder.
5. Now please run with the link like "buyer.test/index.php/home". 
6. Please make sure that you have created the virtual host and also changes windows host file.
